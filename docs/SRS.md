# Software Requirements Specification (SRS)
# English 60 Days Coach

## 1. Introduction
### Purpose
Ứng dụng web SPA hỗ trợ học tiếng Anh 60 ngày cho người mới bắt đầu.

### Scope
- Vocabulary
- Grammar
- Listening
- Reading
- Speaking
- Writing
- Progress Tracking
- Resource Aggregation

## 2. Stakeholders
- Learners
- Teachers
- Content Managers
- Developers

## 3. Functional Requirements
### FR-01 Dashboard
### FR-02 Learning Plan
### FR-03 Vocabulary + SRS
### FR-04 Grammar Lessons
### FR-05 Listening Practice
### FR-06 Reading Practice
### FR-07 Speaking Practice
### FR-08 Writing Journal
### FR-09 Quiz Engine
### FR-10 Progress Analytics

## 4. Non-functional Requirements
- Responsive
- Offline Support
- PWA
- First Load < 3s

## 5. Data Model
User, Progress, Vocabulary, Notes, Resources

## 6. Acceptance Criteria
- Hoàn thành 60 ngày học
- Theo dõi tiến độ chính xác
- Hoạt động offline
